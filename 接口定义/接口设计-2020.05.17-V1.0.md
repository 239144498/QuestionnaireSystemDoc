## 用户注册API

Url：/api/v1/register/

方法：POST

传入参数：JSON
	
	{
	    email:not null
	    usrname:not null
	    birth:not null
	    pwd:not null
		email_code:not null
	    sex : 0 for female, 1 for male
	}

返回参数：

成功：200 OK

失败：403 Forbidden

错误信息：JSON

	{
	    msg: 
		1 for 邮箱已注册
	    2 for 邮箱验证码错误
		3 for 密码验证错误
		100 for 信息缺失
	}
